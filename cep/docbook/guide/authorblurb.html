Description

A short description of an author.

Processing expectations

Formatted as a displayed block. Sometimes suppressed.

Future Changes

In DocBook V4.0, AuthorBlurb will be removed from some of the places in which it now occurs. Instead of appearing inside Author, for example, a new wrapper element will be created to hold Author, AuthorBlurb, and Affiliation.
