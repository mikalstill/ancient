Common attributes

Parameter Entities

The following parameter entities contain invpartnumber:

Parameter Entities
%bibliocomponent.mix;    

Description

An InvPartNumber identifies a number (an “inventory part number”) in some organization-specific numbering scheme.

Processing expectations

Formatted inline. Sometimes suppressed.

DocBook does not control, or specify, the numbering scheme used by an InvPartNumber. It is likely that this number uniquely identifies the document within the organization that assigns the numbers.
