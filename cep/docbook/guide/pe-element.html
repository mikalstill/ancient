%*.element; Parameter Entities

Name

%*.element; Parameter Entities — Control individual element declarations

Synopsis

Used to control marked sections around the declarations of individual elements.

Description

The %*.element; parameter entities provide marked sections around individual element declarations. You can selectively include or remove element declarations from DocBook by changing these parameter entities.

The following sections identify all of the element parameter entities in DocBook.
