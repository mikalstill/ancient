Numeric and Special Graphic Character Entities (%isonum;)

Name

%isonum; — Numeric and Special Graphic Character Entities

Synopsis

The %isonum; parameter entity includes the ISO character entities with the public identifier: 

  ISO 8879:1986//ENTITIES Numeric and Special Graphic//EN

Description

The following character entities are defined in this entity set:
