lhs

Name

lhs — The left-hand side of an EBNF production

Synopsis

Mixed Content Model

lhs ::=
(#PCDATA)

Attributes

Common attributes

Description

This element is only available if you are using the EBNF Module.

The left-hand side (LHS) of a production is a non-terminal defined in terms of the right-hand side (RHS) of the Production.

Parents

These elements contain lhs: production.

Examples

For examples, see productionset .
