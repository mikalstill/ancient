Alternative Greek Symbols Character Entities (%isogrk4;)

Name

%isogrk4; — Alternative Greek Symbols Character Entities

Synopsis

The %isogrk4; parameter entity includes the ISO character entities with the public identifier: 

  ISO 8879:1986//ENTITIES Alternative Greek Symbols//EN

Description

The following character entities are defined in this entity set:
