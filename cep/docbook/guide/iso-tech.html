General Technical Character Entities (%isotech;)

Name

%isotech; — General Technical Character Entities

Synopsis

The %isotech; parameter entity includes the ISO character entities with the public identifier: 

  ISO 8879:1986//ENTITIES General Technical//EN

The following character entities are defined in this entity set:
