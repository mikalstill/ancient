Box and Line Drawing Character Entities (%isobox;)

Name

%isobox; — Box and Line Drawing Character Entities

Synopsis

The %isobox; parameter entity includes the ISO character entities with the public identifier: 

  ISO 8879:1986//ENTITIES Box and Line Drawing//EN
