setErrorHandler

[static]

Adds a global cepErrorHandler which will be used to display and log errors. If no error handler is installed then display and log are not performed

Parameters:
hthe error handler to be used by all cepErrors

 ~cepError ()

~cepError

Destructor. if this is called and the error has not been actioned then the error is logged. Note that if no error handler is installed, failure to action will not be detected
